    XGJ24 Fall Snakes On The Plane

    Link to Itch.io page where you can download the game: XGJ24 Fall Snakes On The Plane

    1. Description

    Snakes On The Plane was made during XGJ24 Fall. My team was me, myself and I. Theme was a "Horror" and "Socket". Game was done in time and I am planing to continue developing it further. Plan is to get the settings work and learn more how the UIToolkit works in Unity.

    2. Description of functions

    Gameplay itself is very simple, there is three (3) pressureplates which can be put down. There isn't much of the gameplay because my plan was to learn UItoolkit and how it works. UItoolkit is very usefull tool to make UI's in Unity. UIToolkit uses UXML and USS codes wich are almost the same as XML and CSS.

    I didn't have enough time to make the settings work for the jam build but I will finnish them on the other project time. This is because I have a nice base done in the Snakes on the Plane and I would like to explore more of UIToolkit.
